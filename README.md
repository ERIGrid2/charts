# ERIGrid 2.0 Helm Charts

The ERIGrid 2.0 Helm charts are a collection of applications packaged for easy depoyment on a Kubernetes cluster.
## Usage

```bash
helm repo add erigrid2 https://erigrid2.github.com/charts/
helm repo update
```
